# App Store — "What's New" (iOS 1.0.0)

Paste into App Store Connect → this version → What's New. This is a big catch-up
release for iOS (coming from 0.2.0), so it bundles weeks of improvements.

Plain-text version (App Store field):

---

Thanks for keeping your collection in Tarantuverse — this is a big update!

• Bigger care-sheet library: over 320 species across 9 groups — tarantulas, scorpions, centipedes, mantises, true spiders, millipedes, vinegaroons, whip spiders, and roaches — each with husbandry details, temperament, and honest safety/venom notes.
• Browse the full catalog: the species database now loads every species, with search and beginner/intermediate/advanced filters.
• Smoother typing everywhere: the keyboard no longer covers the text field you're using.
• Messaging fixes: the conversation menu (Report, Block, View Profile) works again, and chat headers are cleaner.
• Collection upgrades: every animal counts toward your totals (not just tarantulas), feeding-status badges on the list, tidier rows, and quick taxon filters.
• Calendar date pickers on add/edit and log screens — no more manual date typing.
• Stay current easily: pull the latest version anytime from Settings → Check for Updates.
• The Profile tab is now "Settings," plus lots of polish and bug fixes throughout.

Found a bug or have a request? Reach us from Settings → Contact Support.
