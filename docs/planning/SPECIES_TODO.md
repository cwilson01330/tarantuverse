## üí° Pro Tips

1. **Copy/Paste Friendly Format:**
   - Keep a text file with commonly used phrases
   - "Appropriately sized prey (1/2 to 2/3 body length)"
   - "coco fiber, peat moss, or potting soil"

2. **Temperature Conversions:**
   - Most sources use ¬∞F (US) or ¬∞C (International)
   - Our form uses ¬∞F (convert if needed)

3. **Common Name Variations:**
   - Include multiple common names
   - "Mexican Red Knee, Mexican Redknee Tarantula"

4. **Care Guide Template:**
   ```
   **[Common Name]** - [1-2 sentence description]

   [Care difficulty summary - why beginner/intermediate/advanced]

   [Behavior notes - docile/skittish/defensive]

   [Special considerations - humidity, speed, venom, etc.]
   ```

5. **Verify Taxonomy:**
   - Check World Spider Catalog for current scientific name
   - Many species have been reclassified
   - Use current accepted name
