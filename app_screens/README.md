# App Store screenshots

Source PNGs for the iOS App Store listing, captured for each release.

Current set: **v0.2.0** — submitted 2026-04-24. 5 screenshots at
1320 × 2868 (6.9" iPhone display, e.g. iPhone 16 Pro Max). Apple scales
this set down for smaller iPhone display sizes automatically.

## File map

The numeric prefixes match the upload order in App Store Connect, so
re-uploading is a drag-and-drop in the same order. Re-shoot at the
same resolution for future releases or App Store Connect will reject.

## Replacing for a future release

1. Capture fresh screenshots on a 6.9" iPhone (or 6.9" simulator).
2. Drop into this folder, overwriting numerically.
3. Update the version note above.
4. Upload to App Store Connect → App Store tab → screenshots panel.

The release-process doc lives at `docs/release/APP_STORE_RELEASE_2026-04.md`
with the full shot list, headline copy, and listing fields for reference.
